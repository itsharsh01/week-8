# Week 8 Assignments

This repository contains daily work for Week 8 of the PG Diploma AI-ML and Agentic AI Engineering program.

## Folder structure
- `monday/`: Monday assignment notebooks and resources
- `tuesday/`: Tuesday assignment files, including data cleaning + NumPy neural network work

## Tuesday quick links
- `tuesday/tuesday_assignment.ipynb`: main submission notebook
- `tuesday/README.md`: detailed Tuesday setup and run instructions
- `tuesday/prompts.md`: AI prompt log and critique

## Environment
- Python 3.11+ recommended
- Install dependencies as needed per day folder README

Example install (Tuesday):

```bash
pip install numpy pandas matplotlib scikit-learn kagglehub ucimlrepo nbformat
```

## Notes
- Keep day-wise files inside their respective folders.
- Avoid committing temporary artifacts such as `__pycache__` and `.ipynb_checkpoints`.
